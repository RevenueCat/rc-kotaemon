Offering metadata example use cases

Offering Metadata can be utilized to create more flexible paywall experiences, target specific users with particular offerings, or make updates on the fly. Below, we'll show some basic and advanced examples of how you can utilize offering metadata to change out the copy and button color, change the paywall description based on device language, and show a completely different paywall to customers who opened your app via a deeplink.

With the right setup, you can utilize this feature to make changes dynamically without having to push out a new app release every time youâ€™d like to change something!
